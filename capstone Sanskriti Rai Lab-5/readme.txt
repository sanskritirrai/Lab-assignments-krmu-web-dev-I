-- Project Title & Theme

PayFlex – Seamless Online Payments
A responsive landing page showcasing PayFlex as a secure and fast digital payment solution.

-- Sections Implemented

Navbar with anchor links
Hero section with headline, button, and image
Trusted By logos
Features cards
Pricing plans
Testimonials
Newsletter signup section
Contact form
Footer with hover links

-- Breakpoints Used

Desktop: ≥1024px
Tablet: ≥768px
Mobile: <768px

-- Main Layout Techniques
Flexbox layout, responsive units, media queries, hover effects, and colored background sections.

-- How to Open Locally
Download the project folder
Make sure main.html and main.css are in the same directory
Open main.html in any browser
